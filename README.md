# C++ Template 延伸自主練習：學生成績管理系統

這是一個基於 C++17 標準開發的學生成績管理系統，應用了 C++ 核心的 Template 概念以及 Standard Template Library (STL) 容器與演算法。

## 系統環境與編譯要求
- 編譯器：支援 C++17 的 GCC (g++)
- 開發環境：Linux / Windows (MinGW) / macOS

## 編譯與執行方式
請在終端機中確保已安裝 g++，並在檔案所在目錄執行以下指令：

```bash
# 1. 編譯程式 (確保使用 c++17 標準)
g++ -std=c++17 main.cpp -o main

# 2. 執行程式
./main    # Linux 或 macOS 環境
main.exe  # Windows 環境
